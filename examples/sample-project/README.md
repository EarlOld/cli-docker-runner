# Example Project for Testing

This is a sample project to test cli-docker-runner.

## Setup

1. Install dependencies:
```bash
docker-runner install
```

2. Run the project:
```bash
docker-runner run
```

3. Select a script when prompted
4. Configure environment variables

## Available Scripts

- `start` - Start the development server
- `build` - Build for production
- `test` - Run tests
